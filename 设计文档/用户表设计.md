# 表设计

## 用户表设计User

| 字段名          | 类型   | 备注         |
| --------------- | ------ | ------------ |
| id              | Long   | 主键id       |
| username        | String | 用户名       |
| nickname        | String | 用户昵称     |
| password        | String | 用户密码     |
| email           | String | 邮箱         |
| registerIp      | String | 用户注册ip   |
| registerAddress | String | 用户注册地址 |
| activateTime    | Date   | 最近活跃时间 |
| createTime      | Date   | 用户创建时间 |
| updateTime      | Date   | 用户更新时间 |
| isDeleted       | int    | 是否删除     |
